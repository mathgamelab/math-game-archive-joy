# Play Math Archive 🎮📐

**현직 교사들이 만든 디지털 수학 콘텐츠 아카이브**

🌐 **사이트**: [mathgame.kr](https://mathgame.kr)

게임을 통해 수학을 재미있게 배우고, 추상적인 개념을 눈과 손으로 익힐 수 있는 인터랙티브 학습 플랫폼입니다. 중·고등 수학은 물론, 사고력을 키우는 뇌풀기 게임과 교사용 도구도 함께 모아 두었습니다.

| 메뉴 | 주소 | 설명 |
|------|------|------|
| **메인** | [mathgame.kr](https://mathgame.kr) | 소개, 추천·인기 게임, 플랫폼 특징 |

| **수학 게임 빌더** | [mathgame.kr/edugame_builder](https://mathgame.kr/edugame_builder) | AI로 수학 게임 아이디어·기획·프롬프트 생성 (교사용) |

개별 게임은 카탈로그에서 **플레이** 버튼을 누르면 새 창에서 바로 실행됩니다.

---

## 주요 특징

- **교육과정 기반** — 핵심 개념과 성취기준에 맞춰 기획된 콘텐츠
- **게이미피케이션** — 연산, 퍼즐, 전략 게임으로 개념을 자연스럽게 익힘
- **교과별 분류** — 중학·고등·뇌풀기·학급운영 탭으로 정리
- **난이도·검색·필터** — 원하는 게임을 빠르게 찾기
- **플레이 통계** — 인기 게임·플레이 횟수 반영
- **모바일 안내** — 카드에 모바일 지원 여부 표시
- **무료 이용** — 별도 가입 없이 브라우저에서 바로 플레이

---

## 교육 과정

### 중학교 수학
- **1학년** — 정수, 유리수, 문자와 식, 일차방정식
- **2학년** — 연립방정식, 일차함수, 확률과 통계
- **3학년** — 이차방정식, 이차함수, 원의 성질

### 고등학교 수학
- 공통수학, 대수, 미적분 I·II, 확률과 통계, 기하, 인공지능 수학

### 기타
- **뇌풀기 게임** — 사과 게임, 틱택토, 동물장기 등 창의·전략 사고
- **학급운영 도구** — 수업·학급 운영에 활용 가능한 미니 도구 (순차 추가 예정)

## 플레이 가능한 게임 예시

| 게임 | 학습 내용 |
|------|-----------|
| 넘버 플로우: 정수 | 정수 혼합계산 |
| 사과 게임 | 연산력 |
| 도형 탐지기 | 입체도형·단면 |
| Math City | 좌표·함수 개념 |
| 경우의 수 챌린지 | 경우의 수·순열·조합 |
| 얼티밋 틱택토 | 전략·논리 (뇌풀기) |
| 3D 틱택토 | 공간 지각 (뇌풀기) |

카탈로그에는 **플레이 가능**과 **개발 중** 게임이 함께 표시됩니다. 개발 중인 항목은 곧 공개될 콘텐츠입니다.

---

## 이용 방법

1. [mathgame.kr/games](https://mathgame.kr/games) 접속
2. 상단 탭에서 **중학수학 / 고등수학 / 뇌풀기** 선택
3. 학년·과목 서브탭과 **검색·난이도 필터**로 게임 찾기
4. 카드 클릭 → 상세 정보(성취기준, 학습 용어 등) 확인
5. **플레이** 버튼으로 게임 시작

> 💡 대부분의 게임은 **PC·태블릿** 이용을 권장합니다. 모바일 제한 게임은 카드에 표시됩니다.
