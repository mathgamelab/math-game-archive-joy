📋 기본 규칙

  • • 각 플레이어는 3장의 카드를 선택
  • • 선택한 카드들로 다항식을 완성
  • • 높은 점수를 얻은 플레이어가 라운드 승리
  • • 3라운드 중 2승을 먼저 달성하면 최종 승리

🏆 점수 계산

  • • 기본 점수: 차수 × |계수|
  • • 동류항 보너스: +2점
  • • 예시: 2x² + 3x → (2×2) + (1×3) = 7점
  • • 동류항 예시: x² + 2x² → 3x² = 6점 + 2점 = 8점
