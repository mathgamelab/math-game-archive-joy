📖 게임 규칙

  • ✓ 흰돌(⚪)과 검은돌(⚫)이 번갈아가며 진행합니다
  • ✓ 흰돌은 왼쪽(2,3사분면), 검은돌은 오른쪽(1,4사분면)에 배치됩니다
  • ✓ 자신의 차례에 돌을 선택하고 기울기를 정한 후 힘을 조절해서 발사합니다
  • ✓ 일직선의 경로를 따라 돌이 움직입니다
  • ✓ 상대방의 돌과 충돌하면 둘 다 튕겨나갑니다
  • ✓ 게임판 밖으로 나간 돌은 제거됩니다

🎮 조작 방법

  1. 1단계: 게임판에서 자신의 돌을 클릭해서 선택합니다
  2. 2단계: 발사 방향을 선택합니다 (← 왼쪽 / 오른쪽 →)
  3. 3단계: 기울기(m)를 입력합니다 (예: 2, 1/2, -3/4)
  4. 4단계: 힘 게이지를 클릭해서 멈춤으로 발사 강도를 정합니다

🏆 승리 조건

상대방의 모든 돌을 게임판 밖으로 제거하면 승리합니다!
